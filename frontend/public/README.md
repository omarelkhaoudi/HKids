# Dossier Public - Assets Statiques

## Logo de l'application

Pour utiliser l'image comme logo de l'application HKids :

1. **Placez votre image dans ce dossier** (`frontend/public/`)
2. **Nommez-la** : `logo.png`
3. **Format recommandé** : PNG avec fond transparent ou JPG
4. **Taille recommandée** : 
   - Minimum : 256x256 pixels
   - Optimal : 512x512 pixels ou plus
   - Format carré recommandé pour un meilleur rendu

### Emplacements où le logo apparaîtra :

- ✅ **Page d'accueil** (Home) - Header
- ✅ **Page Admin Login** - Centré en haut
- ✅ **Dashboard Admin** - Header du sidebar

### Si l'image n'est pas trouvée

Le composant Logo utilisera automatiquement une icône de livre en fallback si l'image `logo.png` n'est pas trouvée.

### Exemple de structure :

```
frontend/
  public/
    logo.png          ← Placez votre image ici
    README.md
```

Une fois l'image placée, rafraîchissez votre navigateur pour voir le nouveau logo !
