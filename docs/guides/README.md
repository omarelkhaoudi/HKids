# Guides

Ce dossier contient tous les guides et tutoriels pour utiliser et configurer le projet HKids.

## Contenu

- **QUICK_START.md** - Guide de démarrage rapide
- **SETUP.md** - Guide d'installation et de configuration
- **START_SERVERS.md** - Instructions pour démarrer les serveurs
- **GUIDE_IMAGES.md** - Guide pour gérer les images
- **LOGO_SETUP.md** - Instructions pour configurer le logo
- **CREER_IMAGES_TEST.md** - Guide pour créer des images de test
- **CHECKLIST.md** - Checklist de vérification
- **DOCKER_README.md** - Guide Docker

## Utilisation

Consultez ces guides pour :
- Installer et configurer le projet
- Démarrer les serveurs backend et frontend
- Gérer les ressources (images, logos)
- Utiliser Docker pour le déploiement
