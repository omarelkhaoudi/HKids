# Dépannage (Troubleshooting)

Ce dossier contient les solutions aux problèmes courants rencontrés lors du développement.

## Contenu

- **RESTART_BACKEND.md** - Instructions pour redémarrer le serveur backend

## Utilisation

Si vous rencontrez un problème :
1. Consultez d'abord ce dossier
2. Cherchez le problème similaire dans les fichiers
3. Suivez les instructions de résolution

## Problèmes courants

- **Serveur backend ne démarre pas** → Voir `RESTART_BACKEND.md`
- **Rate limiting (429)** → Le rate limiting a été ajusté pour le développement (50 requêtes/5 min)
